## Requisitos
- Ter [**Git**](https://git-scm.com/) para clonar o projeto.
- Ter [**Node.js**](https://nodejs.org/en/) instalado.

<br/>

----------


## Techs

- [x] React.js
- [x] Styled Components
- [x] TypeScript

<br/>


## Starting Dev Environment

<br/>

```javascript

1. Run `npm install` or `yarn install`.
2. Run `yarn start` and access `http://localhost:3000`.

```
